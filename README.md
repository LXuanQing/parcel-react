# parceljs-react
最简单的例子

## 使用前需要配置.babelrc文件
`   {
        "presets": [
            "react-app"
        ]
    }
`

## 环境准备
** 安装 react react-dom babel-preset-react-app ** 

```sh
npm i react react-dom babel-preset-react-app -S
```
** 安装 parcel-bundler **

```sh
npm install -g parcel-bundler
```

### 常用命令
**启动项目**
```sh
npm start 
```
**打包项目**
```sh
npm run build 
```
